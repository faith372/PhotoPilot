# PhotoPilot AI 项目计划表

## 产品目标

做一个门槛尽可能低的 AI 照片工作台：用户导入照片后，软件自动评分、推荐保留/待定/淘汰，并支持一键调色、美颜、自然语言修图和批量导出。

## 低门槛设计原则

- 打开即用，第一屏就是照片工作台。
- 用户不用理解专业术语，也能用“清透一点”“自然美颜”“别太假”完成修图。
- 专业参数保留，但不放在最显眼的位置。
- 默认启用本地隐私模式，接入云端 AI 前明确提示。
- 每个 AI 结果都给出理由，减少黑箱感。
- 所有编辑都不覆盖原图。

## 下一步执行建议

1. 接入一个真实视觉评分 API，让“AI 分数”和“评分理由”从模拟变成真实结果。
2. 增加 Canvas 导出能力，把当前滤镜参数真正烘焙到图片文件中。
3. 增加 API 设置页，允许用户填入自己的 API Key。
4. 加入批量应用：把一张照片的调色方案同步到同组照片。
5. 再升级为 Electron/Tauri 桌面软件。
